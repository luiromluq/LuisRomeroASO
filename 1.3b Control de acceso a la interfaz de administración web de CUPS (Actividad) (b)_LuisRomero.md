# 1.3b Control de acceso a la interfaz de administración web de CUPS (Actividad) (b)

## Permitir que se pueda acceder a la interfaz web desde cualquier ordenador de la red.

Para permitir que cualquier equipo acceda a:

    http://192.168.56.10:631

Modificamos el bloque principal de acceso:

    <Location />
      Require all granted
    </Location>

<img width="344" height="79" alt="image" src="https://github.com/user-attachments/assets/7bf15e82-480d-4127-afcc-d2cc6ff6dc17" />

## Habilitar la interfaz web sólo en modo local o desde una dirección IP concreta.

### Acceso solo local (localhost)

    <Location />
      Require local
    </Location>

<img width="329" height="82" alt="image" src="https://github.com/user-attachments/assets/0f7d8ae4-1e2c-4c1f-8aca-89646687d41d" />

De esta forma solo se permite:

    http://localhost:631

### Acceso únicamente desde una IP concreta

Ejemplo: permitir solo al cliente 192.168.56.11:

    <Location />
      Require ip 192.168.56.11
    </Location>

<img width="319" height="86" alt="image" src="https://github.com/user-attachments/assets/0ba8815a-396b-41f5-9ec5-595b4bc5512e" />

De esta forma, ningun equipo de la red podrá acceder, solo el que tenga esa IP.

## Permitir acceder a toda la interfaz de administración al usuario pepet, al grupo de profesores sólo para la administración de trabajos de impresión ya cualquier usuario correctamente identificado para gestionar las impresoras.

### Acceso completo al usuario pepet

Editamos:

    <Location /admin>
      AuthType Default
      Require user pepet
      Order allow,deny
      Allow from all
    </Location>


<img width="379" height="138" alt="image" src="https://github.com/user-attachments/assets/26559dac-1545-4163-a608-a20252fe3bb8" />

Creamos un usuarios y le damos privilegios:

    sudo useradd pepet
    sudo passwd pepet
    sudo usermod -aG lpadmin pepet

### Acceso al grupo “profesores” solo para la administración de trabajos

Editamos:

    <Location /admin/jobs>
      AuthType Default
      Require group profesores
      Order allow,deny
      Allow from all
    </Location>

<img width="298" height="139" alt="image" src="https://github.com/user-attachments/assets/b9990780-d3ff-494b-937b-7e68f06016af" />

Crear grupo y añadir usuarios:

    sudo groupadd profesores
    sudo usermod -aG profesores usuario1

<img width="515" height="37" alt="image" src="https://github.com/user-attachments/assets/81653126-5a2b-441b-b24f-b1322384b703" />

### Permitir a cualquier usuario autenticado gestionar impresoras

Editamos:

    <Location /admin/printers>
      AuthType Default
      Require valid-user
      Order allow,deny
      Allow from all
    </Location>

<img width="298" height="139" alt="image" src="https://github.com/user-attachments/assets/d1c2745b-bff2-4bee-9a50-57ca3672f4a1" />

Esto obliga a iniciar sesión, pero permite la administración de impresoras a cualquier usuario del sistema.
