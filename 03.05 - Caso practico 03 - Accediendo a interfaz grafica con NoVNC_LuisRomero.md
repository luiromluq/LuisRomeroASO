# 03.05 - Caso practico 03 - Accediendo a interfaz grafica con NoVNC

## Introducción

Ennn el trascurso del curso hemos comentado que no era sencillo utilizar aplicaciones gráficas usando
Docker. En este ejemplo pondremos en marcha un servicio VNC junto con un cliente NoVNC (Cliente para VNC en HTML5 y JavaScript) servido vía web.
Más información sobre VNC y NoVNC en https://es.wikipedia.org/wiki/VNC y en https://novnc.com/.

## Preparando el contenedor

Para este caso práctico, simplemente creamos un contenedor a partir de la imagen descrita en
https://github.com/theasp/docker-novnc, la cual tiene todo listo para funcionar.
La lanzaremos con el siguiente comando:

    docker run --rm -it -p 8080:8080 theasp/novnc

<img width="818" height="554" alt="image" src="https://github.com/user-attachments/assets/477d468d-030b-4e1e-8236-4bf0973d48b3" />

Como es una prueba, hemos creado la imagen con el parámetro “--rm”, por lo cual al cerrar la terminal el contenedor se borrará.

## Accediendo a la interfaz gráfica

Abriendo un navegador a la siguiente URL http://localhost:8080/vnc.html se nos cargará un cliente
NoVNC (Cliente VNC en HTML5 y Javascript). Simplemente, conectaremos con dicho cliente y en
nuestro navegador veremos una interfaz gráfica dentro de nuestro contenedor.
