# Instalación SO

1. Decidimos usar Ubuntu Server por su versatilidad y compatibilidad con el proyecto y en nuestro servidor.

2. Elegimos Ubuntu Server porque es ligero, se puede utilizar de forma intuitiva y permite usar la mayoría de cosas necesarias para nuestro proyecto.

3. Instalamos Ubuntu Server mediante un USB booteable que contenía varios sistemas, además del elegido.

4. Realizamos la instalación de la imagen de Ubuntu Server en el servidor.

5. Configuramos el servidor en forma de interfaz gráfica.

6. Decidimos usar la interfaz gráfica porque nos aclaramos mejor para trabajar.

7. Configuramos la red y el resto de cosas importantes para el proyecto.
