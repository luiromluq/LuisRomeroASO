# 1.3a Instalación, gestión y configuración de la impresora virtual PDF (Actividad) (a)

## Instalar en CUPS la impresora virtual de documentos PDF.

### Instalación
  
    sudo apt update
    sudo apt install cups-pdf

<img width="1202" height="510" alt="image" src="https://github.com/user-attachments/assets/04d6bfc5-d97a-442b-b9e1-e89c8c7c79c8" />

### Activar y reiniciar CUPS

    sudo systemctl enable --now cups
    sudo systemctl restart cups

<img width="920" height="79" alt="image" src="https://github.com/user-attachments/assets/03c1a07b-0e1d-4812-9a6f-5341761981ec" />

Tras esto, la impresora aparecerá como:

    PDF

o

    Virtual_PDF_Printer

## Averiguar consultando Internet cómo cambiar el directorio por defecto al que la impresora PDF envía los trabajos de impresión.

La ruta se cambia en el archivo:

    /etc/cups/

<img width="989" height="74" alt="image" src="https://github.com/user-attachments/assets/f77f624e-b1e8-4e73-b0e8-c6f07a399e7c" />

Edita:

    sudo nano /etc/cups/cups-pdf.conf

<img width="1276" height="799" alt="image" src="https://github.com/user-attachments/assets/4e9b275b-9195-4b78-ad1f-73ed0d51f3ab" />

Busca la línea:

    Out ${HOME}/PDF

Para cambiar la carpeta, por ejemplo a /home/tu_usuario/Documentos/PDFs:

Modifica:

    Out /home/tu_usuario/Documentos/PDFs

<img width="1276" height="803" alt="image" src="https://github.com/user-attachments/assets/ac96e04b-e862-4cf2-b8d9-a70a2780f694" />


Guarda, cierra y reinicia CUPS:

    sudo systemctl restart cups
