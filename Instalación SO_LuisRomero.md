# Instalación SO

1. Hemos seleccionado Ubuntu Server como sistema operativo para el servidor debido a su estabilidad, rendimiento, compatibilidad con los servicios requeridos y amplio soporte por parte de la comunidad.

2. La elección de Ubuntu Server se ha basado en que es una distribución ligera, segura y eficiente, además de facilitar la administración de servicios de red y aplicaciones necesarias para el desarrollo del proyecto.

3. Creamos medio de instalación mediante un USB booteable que contenía la imagen ISO de Ubuntu Server.

4. Posteriormente, se realizamos la instalación del sistema operativo en el servidor siguiendo la configuración recomendada para entornos empresariales.

5. Una vez finalizada la instalación, se configuró el entorno gráfico para facilitar las tareas de administración y mantenimiento del sistema.

6. Optamos por utilizar una interfaz gráfica debido a que permite una gestión más cómoda e intuitiva de los diferentes servicios y configuraciones del servidor. También porque se requeria.

7. Finalmente, configuramos los parámetros de red, usuarios, permisos y demás aspectos necesarios para garantizar el correcto funcionamiento de la infraestructura del proyecto.
